# Prettify JSON

Срипт позволяет распарсить и вывести в консоль json в красивом формате.

# Quickstart

Example of script launch on Linux, Python 3.5:

```bash

$ python pprint_json.py
# После чего нужно указать путь до файла на локальной машине

```
