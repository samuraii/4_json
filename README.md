# Prettify JSON

Скрипт позволяет красиво отобразить json файл в консоль

# Quickstart

Пример работы скрипта для json-файла с содержанием: 

```json
[{"key":"value","key2":value}]
```

```bash
$ python pprint_json.py <path to file>

{
    "key": "value",
    "key2": "value"
}

```
